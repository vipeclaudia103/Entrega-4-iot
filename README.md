## Instalar InfluxDB 2
Instalar InfluxDB 2.0 como servicio:

```bash
# Ubuntu/Debian AMD64
curl -O https://dl.influxdata.com/influxdb/releases/influxdb2_2.7.5-1_amd64.deb
sudo dpkg -i influxdb2_2.7.5-1_amd64.deb

```

Iniciar el serrvicio

```bash
sudo service influxdb start
```
